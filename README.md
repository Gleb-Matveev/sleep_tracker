### Domains
- User - self explanatory.
- Goal - a discription of a goal user is trying to reach by using this app (in regard to his sleeping habits/schedule).
- Routine - a routine user follows in order to achieve his goals.
- Rule - a rule user follwos in order to achieve his goals.
- Day - one statistic entry with information on sleep duration and quality.
